What one run produces

  • KPI cards — revenue, orders, on-time %, low-stock count, each vs last week and an 8-week baseline
  • Findings with named drivers, and SPC signals separated from week-to-week noise, with the arithmetic shown
  • A data-quality gate and source reconciliation — the report audits its own inputs
  • A Power BI Command Center (TMDL + PBIR, authored as code) from the same guarded extraction

Read-only, enforced in code

  • A four-layer guard: lexical, parse-tree, a side-effecting-function check, and a read-only session
  • The guard checks the functions a statement calls, not just its shape — and fails closed when it can't parse one
  • Three Turkish ERP profiles (Logo Tiger, Netsis, Mikro) plus a guarded MCP server for agent access
